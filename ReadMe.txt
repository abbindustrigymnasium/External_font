Title: 
CSS practise with surfing theme

Description: 
A CSS project for a surfing store.
Only the head of index.html was allowed to be edited.
The design was done using two CSS files.

Installation: 
Use the extension yandeu.five-server in Visual Studio Code.
The extension normalize was used.

Usage: 
Right click on the index.html file and select "Open with Five Server".
